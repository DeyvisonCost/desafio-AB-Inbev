# **Nest.JS - Documentation**

> ## Pré-requisitos
>
> Node.js v18 ou superior
> Yarn (ou npm)

### Instalação:

1 - Instale as dependências:
`yarn`

2 - Rode o servidor de desenvolvimento:
`yarn start:dev`

3 - O projeto estará disponível em:
`http://localhost:3334`
